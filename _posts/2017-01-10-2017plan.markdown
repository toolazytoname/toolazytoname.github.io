# 0 前言<a name="preface"></a>

现在的我，坐在从霾都开往东海曙光之滨的列车上，在目的地有我的妻子和女儿在等着我。最近在路上的时间特别多，我还喜欢自己一个人的旅途，可以看点东西，没人打扰。累了，可以睡会儿，还是比较惬意的。现在顺带着就可以把年终总结给写一下。去年写了一篇《写在猴年伊始》，看看去年那时候自己的想法，计划，目标。正好现在差不多也有一年了。不是有个段子说，新年的计划是去年想要完成的前年计划。可以复复盘，看看自己实现了哪些。


# 1 最大的收获<a name="gain"></a>

最大的收获，是我的小伊伊啦。多和伊伊接触，陪伴。科学喂养，原来去年肚子里面的就是你啊，因为想顺产，一直控制体重。就结果看，体重控制的很完美，2.9千克。但是你太调皮了，脐带绕颈三周，胎位又是臀位，这就算想做调整胎位的体操，也做不了。只能剖腹产，心疼死我了。我推断估计是一个很调皮的儿子。谁知道是这么调皮好动的是一个女儿，不愧是我的女儿啊。据说，我当年也是本来好好的头位，临生产又拧过去了。还有最近妈妈反溃，你脾气很大，易怒，这不会也是像我吧，难道我内心深处其实是这样子的。不管如何，新年最重要的是，把你和你妈妈照顾好。

# 2 游戏<a name="game"></a>

去年在游戏上投入了不少的时间，玩玩《炉石传说》、《皇室战争》、《部落冲突》、《极品飞车》，还有WiiU上的一些。新的一年尽量就少玩，WiiU上的可以继续尝尝鲜。通过玩游戏，得到的感触是，游戏把人性研究的很透。会制定一个明确的目标，让我去完成，完成以后马上就可以获得相应的奖励。反馈很实时，不像我背了几个月的单词，但是我的英语听说读写的能力，并不是增长得那么立竿见影。游戏能马上让我爽到，我换了一个策略，马上试一把，赢了很有成就感，会再接再厉。输了不服，赌徒心理，下一把要赢回来。因此我在制定今年的目标的时候，会尽量量化，同时尽量让自己多尝尝甜头，产生成就感。另外我很喜欢游戏里面可以搭出很多不同的组合，不同的效果直接合作，给我很大的想象空间，很有意思。

# 3 健身<a name="health"></a>

健身方面，去年过年期间，因为脚崴了，那是之前的老伤，长跑就没有坚持下去，打算等好利索了以后，再继续长跑。虽然荒废了也近一年，体重也持续增长，但是精神面貌自我认为还是不错的，有点吃老本了啊，哈哈。新年目标标准引体向上，正握10次，反握10次。标准俯卧撑50次。跳绳熟练现已掌握的花式。不喜欢把体重当成一个指标，希望肚子上不要像现在一样这么多赘肉吧，控制饮食，有氧锻炼，减赘肉。从这点，可以看出自己的自制力。

想王石那样登顶过两次珠峰也很cool，保持一项全国纪录16年16天也cool，但是之所以觉得酷，是因为人家首先是一个有钱人，对吧。女朋友也年轻漂亮，按年龄差不多可以当女儿了。

# 4 音乐<a name="music"></a>

重新捡起了我喜欢的口琴，之前吹的都是复音口琴。这回是布鲁斯口琴，觉得自己乐理不太熟悉。希望随着我口琴技艺的提升，对基本乐理，能有更全面的了解。现在只会简单的《欢乐颂》《送别》《down by the sally gardens》《oh susanna》。还没有学到压音和超吹。自己给自己定的最终目标当然是《卡门》了，目前了解。希望新的一年，学会压音和超吹，至少把每个音都能吹出来，这样能吹的歌就更多了。同时把握好节奏，按拍子吹。


# 5 英语<a name="english"></a>

按照目前的进度，每天一个小时多点，差不多，四月初我就可以背完考研六千多个词汇，用的是《百词斩》里面的每日打卡和单词电台。同时结合，喜马拉雅，里面的《铅笔英语》《虐儿万词》之类的，新年目标定在2W词汇量，即使搞定1W，对现在的我来说也是很大的进步。毕竟初中毕业后就没有正紧学过英语。希望我再看下一季《权利的游戏》《西部世界》的时候，就不需要看字幕了，当然像《破产姐妹花》这种语速快，俚语多的应该没那么快听懂。《权利的游戏》很多都是演讲性质的，吐字还是比较清晰的。玩游戏的时候，也不会不知所云。看文档不要有陌生感和排斥感，毕竟我这么崇洋媚外。啊盼仔裸测都有8k词汇量，要把她给比下去。

关于如何记忆词汇，目前比较侧重词根，词缀，词源。我对逻辑性的记忆比较敏感，数字、字母、人名弱。最近比较迷恋《最强大脑》，很叹服其中选手，所展现出来的，记忆，推理，联想，速算等超乎常人的能力。我记忆单词，也算是对自己记忆力的一种开发，hack自己的大脑。另外我觉得能听懂美剧，看外文原版小说，本身就是一件很cool的事情，我追求这种很cool的感觉。

# 6 投资理财<a name="invest"></a>


几乎全投进了固定资产，新年目前想到的是，可以继续读一点，入门的书籍，培养自己的财商。这个我目前对自己没啥信心和野心，总资产10%的收益，也不一定能达到。希望新年可以多尝试几种方式和方法，探路熟悉游戏规则为主，收益为辅。

# 7 高等数学<a name="higherMathematics"></a>

这个我有点三分钟热度了，还是得看看书啊。直接做题有点费劲了。考验应该是在12月，不知道给自己定在6月份之前，走完一轮，这个目标怎么样。

# 8 阅读<a name="reading"></a>

多读一些有意义的书，去年能回忆起来的应该也就是《白夜行》《解忧杂货铺》《追风筝的人》等杂书。确实《解忧杂货铺》看着很温暖，两本书的逻辑推理都很严密，如果东野圭吾当程序员，逻辑思维一定很严谨。

# 9 专业<a name="professional"></a>
前面乱七八糟的说得这么热闹，正经到这里，就不行了。随着年龄的增长，自我认知自己的专业技能并未增长。先复盘一下，去年完成的几点。

1. 个人博客已经搭建，现在的域名应该快到期了。新年换个.com的域名吧。博客略简陋，可以加个主题，加点评论互动，代码加点样式啥的。
2. Git 的书草草过了一遍，自己整理出一个很大的思维导图。最后一章节关于原理的也没看。但也已经受益无穷了。基本上，平常同事的问题都能搞定。
3. 工作中遇到的一些问题，都有相应的成果整理，或是文章，或是一个类。
除了这三点，好像别的计划都已付之东流，足见我是一个自制力极差的人。
